# WebView

> 这个组件使用了 html 的 iframe 作为内部实现

[[toc]]

:::demo

```vue
<template>
  <div class="device-ios">
    <div class="device-inner" v-loading="loading">
      <ba-web-view
        src="https://vant-ui.github.io/vant/v2/mobile.html#/zh-CN/button"
        @load="onLoad"
        :height="500"
      ></ba-web-view>
    </div>
  </div>
</template>
<script>
export default {
  data() {
    return {
      loading: true
    };
  },
  methods: {
    onLoad() {
      this.loading = false;
    }
  }
};
</script>
```

:::
