# demo-container 组件示例

Demo Container 是一个基于 Vuepress 的插件，它可以帮助你在编写文档的时候增加 Vue 示例，它的诞生初衷是为了降低编写组件文档时增加一些相关示例的难度。

# 安装
pnpm add vuepress-plugin-demo-container -D
1
2
// 配置 .vuepress -> config.js
module.exports = {
  plugins: [['demo-container']]
};
